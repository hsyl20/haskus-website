About us

  • At Haskus, we develop software and web-sites
  • Our domains include:
    • Functional programming
    • System programming
    • High-performance computing
    • Web programming
  • We mainly use the Haskell programming language
  • We use and develop free and open-source software as much as possible
  • We are located in Poitiers (France)

Ongoing developments

  • ViperVM: a framework to perform system programming with only Haskell and the Linux kernel. It aims to be especially useful to OS researchers.
  • drums-school.fr: an online course to practice drums.
  • allons.mobi: a website designed for mobile phones to easily find places and attend events of interest (concerts, open stages, pubs, etc.).

Past developments

  • dynamic-linker-template: Haskell library that uses Template Haskell to easily create a dynamic library loading interface from a record (using field names for library symbols)
